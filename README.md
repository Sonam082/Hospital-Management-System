# Hospital Management System

## Overview

The Hospital Management System is an Excel-based solution built to help healthcare administrators monitor key hospital data, track performance, and make informed decisions. The solution includes a dashboard with charts, KPIs, and visual indicators designed to deliver actionable business insights.

## Key Features

- Centralized hospital performance dashboard
- Interactive charts and visualizations
- Key Performance Indicators (KPIs) for critical metrics
- Data-driven monitoring for departments and resources
- Easy-to-use Excel interface with dynamic reporting

## Project Structure

- `README.md` - Project documentation and usage guidance
- `dashboard.png` - Preview image of the hospital management dashboard
- `ExcelWorkbook/` - Contains the Excel workbook and supporting files for the project


## Benefits

- Improves visibility into hospital operations
- Supports faster decision-making through visual reporting
- Helps identify trends, bottlenecks, and opportunities for improvement
- Designed for hospital administrators and healthcare analysts
